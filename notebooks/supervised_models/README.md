# Supervised Model Notebooks

Presentation-friendly notebooks for common supervised learning models using small built-in scikit-learn datasets.

Included notebooks:

- `linear_regression_diabetes.ipynb`
- `logistic_regression_breast_cancer.ipynb`
- `decision_tree_wine.ipynb`
- `random_forest_wine.ipynb`
- `support_vector_machine_breast_cancer.ipynb`
- `gradient_boosting_breast_cancer.ipynb`
- `naive_bayes_iris.ipynb`
- `kNN.ipynb`

These notebooks are intentionally compact: each one loads data, trains a model, evaluates it, and records a short practical note about when the model is useful.
