Approach

Built to be reviewed quickly by hiring teams and discussed deeply in interviews.

Why scripts first

The repository treats Python scripts as the main implementation layer so reviewers can inspect logic, imports, model setup, and evaluation without notebook noise.

Where notebooks fit

Notebooks remain part of the structure for walkthroughs, visual diagnostics, and presentation-friendly explorations that sit on top of the scripts.
